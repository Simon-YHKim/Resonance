# 제1부. 등장 캐릭터 도감

## 계층 1. 중심 인물 (3종)

세 사람이 잔향 전체의 무게를 진다. **the Named, the Voice, 원의 아이**. 셋은 사실 한 사람의 *세 얼굴*이지만, 게임 안에서는 끝까지 셋으로 분리되어 등장한다.

---

### 1-1. the Named (이름을 가진 사람)

플레이어 본인. 잔향에 들어갈 자격을 가진 단 한 사람.

| 항목 | 내용 |
|---|---|
| 인게임 호칭 | the Named (영문) / 이름을 가진 사람 (한글) |
| 등장 형태 | 2D 타일맵 모드의 아바타 + GPS 모드의 위치 마커 |
| 외형 | LLM이 닉네임 카테고리(v2.4 A~H)에 따라 자동 생성. 외형 바리에이션 약 200종 |
| 닉네임 | 사용자 직접 입력. 게임 전체의 가장 깊은 시드 |
| 4대 키워드 | 사용자 닉네임에 따라 1~2개 자동 매칭 |
| 자아 시기 | *현재의 어른*. 잔향에 들어와 시간을 거꾸로 거슬러 올라간다 |

**잔향에서의 위치**

> the Named는 잔향의 유일한 *외부 진입자*다. 다른 모든 캐릭터(the Voice 포함)는 잔향 안에 *살고 있는* 존재이거나, 잔향 그 자체의 일부다. the Named만이 *깨어나서 돌아갈* 수 있다.

**기억 구조**

| 시기 | 상태 |
|---|---|
| 5~7세 | *원*에 잠들어 있음 (원의 아이) |
| 8~12세 | 흐려져서 잘 기억나지 않음 (떠난 친구들) |
| 13~19세 | 일부 기억하지만 의미는 잊음 (미루는 학자) |
| 20대 | 또렷하지만 *잊으려 노력한* 시기 (흐르는 그림자) |
| 30~40대 | 너무 가까워서 거리가 안 보이는 현재 (남겨진 거인) |

**대사 톤**

the Named의 대사는 사용자가 채팅창에 직접 입력. 단, 캐릭터 생성 시 LLM이 결정한 *기본 톤*이 시스템 프롬프트에 시드로 들어간다. 같은 *"공격"*이라도 the Named마다 LLM 응답이 다른 이유.

### 1-2. the Voice (목소리)

the Named 안에서 어린 자신을 잊지 않으려 남아 있던 수호자 자아. 세계관 바이블 v0.1 §2 참조.

| 항목 | 내용 |
|---|---|
| 인게임 호칭 | the Voice (영문) / 목소리 (한글) — 끝까지 본명 없음 |
| 등장 형태 | 채팅창의 한 줄 텍스트로만 (3장 후반까지) |
| 음성 | 모바일 TTS 기본값. 정확히 *어른인지 어린이인지 가릴 수 없는* 중간 음역 |
| 외형 | 보이지 않음. 종장 직전 흐려지는 시각 효과로만 형태 암시 |
| 화법 | C 혼합형 (평소 산문체 + 결정적 순간 시적 단문) |
| 정체 공개 곡선 | 1장 0% → 2장 20% → 3장 50% → 종장 100% |
| 4대 키워드 | 희생 (자신을 드러내지 않음) + 어린시절 (지켜온 자) |

**4단계 변화 요약**

| 시기 | the Named의 인지 | 화법 변화 |
|---|---|---|
| 1장 | 잔향 안내인 | 차분한 산문체. 단서 3개 박힘 |
| 2장 | *"…나에 대해 너무 잘 아는데?"* | 사라짐 직전 단 한 번 평소 톤을 잃음 |
| 3장 | *"…어디서 들었더라"* | "우리"라는 단어 처음 사용. 어린이 어휘 한 번 흘림 |
| 종장 | *"…!"* (깨달음) | 어휘가 점점 흐려지다 마지막 한 마디 후 사라짐 |

**LLM 구현 사양**

- 모델: Claude Haiku 4.5 전용
- 평균 토큰: 입력 ~600 + 출력 ~30~80
- 1회 비용: ₩0.5~0.8
- DAU 1만 월 비용: ₩45~70만

### 1-3. 원의 아이 (The Child of Origin)

the Named의 5~7세 어린 자신. 잔향의 가장 깊은 곳에 잠들어 있다.

| 항목 | 내용 |
|---|---|
| 인게임 호칭 | 원의 아이 (한글) / The Child of Origin (영문) |
| 등장 형태 | 종장에서만. 3페이즈 보스 (숨바꼭질 → 떼쓰기 → 마지막 부탁) |
| 외형 | the Named 아바타의 1/3 크기. 얼굴은 *the Named가 녹아 흘러내린 듯한* 흐릿함 |
| 옷 | LLM이 캐릭터 생성 시 추출한 어린 시절 시드에서 도출 (줄무늬 티셔츠 / 큰 후드 등) |
| 손에 든 것 | 작은 종이 한 장 — 사실은 빨강·파랑을 그린 그림 (페이즈 2 후반에 밝혀짐) |
| 거주 위치 | 원의 자리 — 회색 운동장. 멀리 등하교 종소리. 다른 아이는 한 명도 없음 |
| 4대 키워드 | 어린시절 (본인) + 추억 (the Named가 잃어버린 시간) |

**원의 아이의 정체 — D 갈래에서**

> 원의 아이는 *the Named의 마지막으로 또렷이 행복했던 한 시점*에 멈춰 선 자아다. 그 이후의 모든 슬픔·실망·외로움이 누적되기 *직전*의 어린 자신. the Named가 어른이 되면서 *그 행복을 지키고 싶어* 무의식이 그 시기의 자아를 *깊은 곳에 잠재워 두었다*. the Voice는 그 자리를 지키며 the Named에게 신호를 보내온 것이다.

**3페이즈 행동 패턴**

| 페이즈 | HP | 메커니즘 | 톤 |
|---|---|---|---|
| 1 숨바꼭질 | 100→60% | 사라졌다 다른 위치로. the Named가 부르면 한 발씩 다가옴 | 호기심·놀이 |
| 2 떼쓰기 | 60→25% | 종이를 보여줌. 무시하면 울음(광역 슬픔 디버프). 봐주면 진정 후 다시 떼씀 | 매달림·불안 |
| 3 마지막 부탁 | 25→0% | 손을 내밂. *"같이 갈래, 아니면 두고 갈래?"* | 화해·받아들임 |

**대사 톤 — 어린이 어휘**

```
[페이즈 1]
"…거기 있어?"
"…한 번 더 부르면, 갈게."
"…잘 왔어요." 
   ※ 정확히 the Voice가 1장 첫 발화로 한 말. 종장에서 사용자가 깨닫는 첫 신호.

[페이즈 2]
"…내 거 봐줘."
"…진짜 잘 그렸는데."
"…왜 안 봐줘…"

[페이즈 3]
"…같이 갈래, 아니면 두고 갈래?"
   ※ 어린이 톤이지만 어딘가 어른의 슬픔이 잠깐 비치는 한 마디.
```

**LLM 변주 — 닉네임 카테고리별 (v2.4 호환)**

| 카테고리 | 변주 |
|---|---|
| A 가족 호칭 | 원의 아이가 the Named를 *그 이름으로* 부른다 ("엄마 어디 있어?" 등) |
| D 위험 단어 | 옷·종이·운동장이 *유난히 또렷*. 그 평화가 the Named에게 더 큰 의미라는 암시 |
| H 안전 + 기타 | 표준 변주 |

## 계층 2. 메인 NPC 동료 (4종)

v2.3에서 4대 키워드와 매칭한 4명의 동료. 1주차 바이블 §3에서 *백스토리는 3주차에 결정*하기로 했으나, 이번에 도감으로 정리할 때 **잠정 시드**를 박아두자. 3주차 작업에서 본격 확장 예정.

각 동료는 the Named의 어떤 *한 면*을 대신 살아온 분신이다. *어른이 되며 the Named가 내려놓아야 했던 자질*이 잔향에서 형태를 가진 것.

---

### 2-1. W (침묵의 W)

| 항목 | 내용 |
|---|---|
| 호칭 | W (단 한 글자. 풀네임 끝까지 모름) |
| 4대 키워드 | 희생 |
| 합류 시점 | 1장 → 2장 전환 직후 임시 / 3장 영구 |
| 외형 | 나이 가늠 어려움. 회색 후드. 두 손 항상 주머니에 |
| 채팅 | *항상 "…"* 만 출력. 단 LLM이 표정·자세·작은 행동을 묘사 |
| 작별 조건 | 4번 보스(떠난 친구들) 처치 시 — 단 한 번 입을 연다 |
| 4번 보스 처치 시 한 마디 | *"…고마워."* (잠정 시드. 3주차에 확정) |
| the Named의 어떤 면인가 (잠정) | *말하지 못하고 묻어둔 것* — 어릴 적 누군가에게 미안하다고 말하지 못한 그 한 마디 |

**행동 패턴**

W는 the Named의 입력에 *직접 반응하지 않는다*. 대신 임베딩 매칭(v2.3 §25.3)으로 자기 행동 카드 30장 중 적절한 것을 고른다. 사용자는 W의 *작은 신호*들을 읽으며 호흡을 맞추게 된다.

```
[전투 중 the Named가 그림자에 숨었을 때]
W가 한 발 옆으로 비킨다. 후드 끝이 살짝 떨린다.
   ※ 채팅: "..." 
   ※ 시스템: W가 "엄호 위치"로 이동. 다음 턴 the Named에게 +방어 버프.

[전투 후 잠시]
W가 the Named 쪽으로 반 발 다가온다. 이내 멈춘다.
   ※ 채팅: "..."
   ※ 호감도 +5
```

### 2-2. 꼬마 R

| 항목 | 내용 |
|---|---|
| 호칭 | 꼬마 R (R은 어떤 이름의 첫 글자인지 끝까지 모름) |
| 4대 키워드 | 어린시절 |
| 합류 시점 | 2장 중반 |
| 외형 | 7세 정도. 늘 우는 듯한 표정. 어딘가 the Named의 어린 시절을 닮음 |
| 채팅 | 어린이 어휘. 짧고 자주 멈춤 |
| 작별 조건 | 3장 끝 — 원의 자리로 돌아감 |
| the Named의 어떤 면인가 (잠정) | *원의 아이의 약한 그림자* — 5~7세보다 살짝 자란, 8~12세 초입의 자아 한 가닥 |

**행동 패턴**

꼬마 R은 *보호 본능을 자극*한다. HP가 낮고 자주 다친다. the Named가 R을 보호하면서 자연스럽게 어린 자신의 취약함과 마주하게 된다.

```
[전투 시작]
꼬마 R: "…나, 무섭지 않아."
   ※ 채팅: 어린이 어휘.
   ※ 시스템: HP 30%부터 자동 *숨기*. the Named가 보호 위치로 이동해야 함.

[보스전 중]
꼬마 R: "…저거, 진짜야?"
   ※ 채팅: 두려움을 어린이 어휘로 표현.
   ※ 시스템: 잊혀진 자 등장 시 R의 공포 디버프가 the Named에게 전이됨. 매크로로 안심시키면 해제.
```

**작별 — 3장 끝**

```
the Voice: "…꼬마 R, 어디로 가는 거예요?"
꼬마 R: "…나, 거기로 돌아갈래."
   ※ 어린이 어휘로 작별 인사.
   ※ R이 회색 운동장 쪽으로 천천히 걸어간다. 사라지기 직전 the Named를 한 번 돌아봄.
the Voice: "…그래도 돼요. 거기 — 곧 다른 아이가 있을 거예요."
   ※ the Voice가 처음으로 *원의 자리에 누가 있는지*를 암시.
   ※ 사용자가 알아채면 강력한 단서. 못 알아채도 됨.
```

### 2-3. 양수(陽守)

| 항목 | 내용 |
|---|---|
| 호칭 | 양수 (한자 "陽守" — 햇빛을 지키는 자) |
| 4대 키워드 | 꿈과 현실 |
| 합류 시점 | 3장 초반 |
| 외형 | 20대 후반 청년. 항상 웃는 얼굴. 옷차림은 단정하지만 어딘가 *너무 단정해서 부자연스러움* |
| 채팅 | 농담 위주. 단 농담 사이에 가끔 *비어 있는 한 줄*이 끼어듦 |
| 작별 조건 | 종장 직전 — 마지막 농담을 남기고 흩어짐 |
| the Named의 어떤 면인가 (잠정) | *진짜 감정을 보이지 못하고 밝은 척했던 자기* — 사회생활에서 항상 웃었던 그 시간들 |

**행동 패턴**

양수는 *항상 분위기를 띄운다*. 전투 중에도, 슬픈 순간에도. 단 *호감도가 임계를 넘으면 잠깐의 침묵*이 길어지기 시작한다 — 가짜 밝음이 점점 무거워지는 시그널.

```
[전투 시작]
양수: "오, 큰 거 왔네! 한 판 해봅시다 형/누나."
   ※ 농담 톤. 호감도 50 미만에서는 거의 항상 농담.

[호감도 80 이상 — 보스전 진입]
양수: "…한 판 해봅시다."
   ※ 농담이 빠진다. 사용자가 알아챈다.
```

**작별 — 종장 직전**

```
양수: "…나는 여기서 빠질게.
       끝까지 따라가도 — 별로 도움 안 될 것 같아.
       
       …있잖아, 나는 사실 — 
       
   ※ 양수가 한 번 멈춘다. 어떤 진짜 말이 나올 것 같다.
   ※ 사용자가 숨을 죽인다.
   
       …재미있었어. 그게 다야."
       
the Voice: "…양수, 거기 잘 있어요."
양수: "어, 형/누나도. 잘 가."
   ※ 양수가 흩어지기 직전 마지막 농담:
양수: "…아, 거기 가서 — 너무 울지는 마라. 어색해."
   ※ 흩어진다. 사용자가 양수의 진짜 감정을 *못 본 채로* 작별이 끝난다.
   ※ 한국 정서의 핵심 — 정말 중요한 말을 끝내 못 하고 헤어지는 것.
```

### 2-4. 노 선생

| 항목 | 내용 |
|---|---|
| 호칭 | 노 선생 (성씨 "노"인지 老(노인)인지 끝까지 모호) |
| 4대 키워드 | 추억 |
| 합류 시점 | 3장 후반 |
| 외형 | 60대~70대 노인. 작고 마른 체구. 손에 항상 작은 수첩 |
| 채팅 | 잔잔한 산문체. 골목·동네·옛이야기 위주 |
| 작별 조건 | 종장 진입 시 — *"이제 너희들끼리 가"* |
| the Named의 어떤 면인가 (잠정) | *어른이 되며 잃어버린 차분함* — 모든 동네에 한 명씩 있던, 시간을 천천히 보던 노인의 기품 |

**행동 패턴**

노 선생은 *길드 시스템의 메타포*다 (v2.3 §25.4). 다른 the Named들의 잔향을 알고 있고, 그 이야기를 the Named에게 들려준다.

```
[추억 거점 재방문 시]
노 선생: "이 자리, 다른 사람도 자주 왔다오. 
         그 사람은 — 혼자 와서 한참 앉아 있었지.
         자네는 뭘 보러 다시 왔나?"
   ※ "다른 사람" = 다른 the Named. 길드 시스템의 자연스러운 빌드업.
   ※ 사용자가 깨닫지 않아도 됨. 종장 직후 길드 탭이 열리며 회상되는 한 마디.

[자유 매크로 신규 등록 시]
노 선생: "…좋은 말씀이오. 그 말, 자네 거 맞나?"
   ※ 사용자가 직접 작성한 매크로에 대한 노 선생의 코멘트.
   ※ "자네 거 맞나" — D 갈래의 핵심 질문이 자연스럽게 등장.
```

**작별 — 종장 진입 시**

```
노 선생: "…여기까지요.
         나는 이 자리가 좋소. 
         자네 — 끝까지 가게."

the Voice: "…노 선생, 잘 계세요."
노 선생: "…잘 가시오. 
         …그리고, 거기서 만나는 그 아이 —
         …내가 안부 전한다고."
         
   ※ 마지막 줄에서 노 선생이 *원의 아이를 알고 있음*을 암시한다.
   ※ 어떻게 아는가? 노 선생도 한때 *그 자리에 다녀온 the Named였기 때문*이다.
   ※ 이 단서가 종장 후 마이페이지 "함께 가는 길" 탭에서 다른 the Named와의 연결로 확장된다.
```

## 계층 3. 잊혀진 자 — 5체 보스 (5종)

v2.3 §24 + v2.4 §28에서 결정된 5체. 각 보스는 the Named의 인생 어떤 시기의 *잊혀진 자아 한 조각*이 형태를 가진 것.

각 보스의 *구체 시드 백스토리*는 2주차 작업이지만, 이번 도감에서 **외형·페이즈·대사 톤은 확정**.

### 3-1. 남겨진 거인 (the Left-Behind Giant)

| 항목 | 내용 |
|---|---|
| 자아 시기 | 30~40대 — 직장·도시 |
| 위치 | 강남역 일대 (서울) / 광역시별 동급 헥스 |
| 외형 | 12층 빌딩 높이. 정장 실루엣. 머리 자리에 *형광등 무더기*. 양손에 빌딩 잔해 |
| 톤 | 압도·허무 |
| 4대 키워드 | 추억 (잃어버린 야망) + 꿈과 현실 (야망과 현실의 거리) |
| 잠정 시드 백스토리 (2주차에 확정) | *the Named가 어떤 직장 사건을 겪고 자신을 너무 작게 느꼈던 그 시기*가 거대한 그림자가 되어 남은 것 |

**4페이즈 패턴**

| 페이즈 | 메커니즘 |
|---|---|
| 1 빌딩 그림자 낙하 | 광역. 회피 헥스 표시. 빌딩이 무너지는 듯한 시각 효과 |
| 2 형광등 깜빡임 | 스턴. 프롬프트 입력 1턴 차단 |
| 3 출근길 군중 소환 | 회색 NPC 다수 출현. AOE로 처치 가능 |
| 4 명함 폭격 | 도트 데미지. 시간 지나면 명함이 흩어짐 |

**대사**

```
[등장]
거인: "…늦었어, 또."
   ※ "늦었다" — the Named가 한때 출근길에 자주 들었거나 자주 한 말.

[페이즈 3 — 군중 소환]
거인: "…여기, 다 너 같은 사람들이지."
   ※ 한국 직장인의 익명성이 풍경에 깔린다.

[처치 후 마지막 한 마디]
거인: "…그래서, 행복했었나요."
   ※ 압도적인 거인이 마지막에 평어체가 아닌 존댓말로 묻는다.
   ※ 잊혀진 자가 적이 아니라 *주인을 기다리는 자*임을 톤으로 박는다.
```

**기억의 조각**

`사원증의 기억` 또는 `엘리베이터의 기억` 등 — LLM이 the Named의 어린 시절 키워드를 시드로 변주.

### 3-2. 흐르는 그림자 (the Drifting Shadow)

| 항목 | 내용 |
|---|---|
| 자아 시기 | 20대 — 첫 사랑·이별 |
| 위치 | 한강 둔치 / 광역시별 강변 |
| 외형 | 강 표면을 따라 길게 늘어진 *두 그림자*. 한 그림자가 천천히 다른 쪽에서 멀어진다 |
| 톤 | 멜랑콜리·아련 |
| 4대 키워드 | 희생 (놓아준 사람) + 추억 (놓치 못한 사람) |
| 잠정 시드 백스토리 (2주차) | *the Named가 한때 깊이 사랑했고 헤어진 누군가*. 또는 *놓치고 후회한 첫 친밀함* |

**4페이즈 패턴**

| 페이즈 | 메커니즘 |
|---|---|
| 1 두 갈래 분열 | the Named가 둘 사이 선택해야 함. 잘못 고르면 큰 데미지 |
| 2 비 | 시야 차단 + 스태미나 회복 정지 |
| 3 강 물결 | 잊혀진 메시지 텍스트가 잠깐 떠올랐다 사라짐. 사용자가 읽으면 약한 디버프 (감정적) |
| 4 소실 | 잠깐 무적. 그동안 the Named의 다음 입력을 *기다린다* |

**대사**

```
[등장]
그림자: "…아직, 보이는구나."

[페이즈 1 — 분열]
그림자 둘이 동시에:
  왼쪽 그림자: "…따라와."
  오른쪽 그림자: "…놓아 줘."
   ※ 사용자는 둘 중 하나를 골라야 한다. 어느 쪽도 정답은 아님.

[페이즈 3 — 강 물결의 텍스트]
"…잘 지내?"
"…미안해."
"…그때, 왜 그랬어?"
   ※ 사용자가 읽지 못하게 빠르게 흘러간다. 멈출 수 없음.

[처치 후 마지막 한 마디]
그림자: "…붙잡았으면 달랐을까."
   ※ 한 그림자만 남고 다른 그림자는 강 끝으로 흘러간다.
```

### 3-3. 미루는 학자 (the Postponing Scholar)

| 항목 | 내용 |
|---|---|
| 자아 시기 | 17~19세 — 고3·시험 |
| 위치 | 종로 학원가 / 광역시별 학원 밀집 지역 |
| 외형 | 얼굴 없는 교복 형체. 얼굴 자리에 *흰 답안지*. 책상 다리가 거미처럼 뻗어 있음 |

| 잠정 시드 백스토리 (2주차) | *그 시험·그 답안지·그 결과* |

**5페이즈 패턴**

| 페이즈 | 메커니즘 |
|---|---|
| 1 시험지 회오리 | 광역. 매크로 사용 1회 봉인 |
| 2 형광펜 표식 | 단일 대상 디버프 |
| 3 종소리 카운트다운 | 3턴 내 처치 못 하면 즉사 |
| 4 답안 제출 | 즉시 큰 데미지지만 후 무방비 1턴 |
| 5 (호감도 낮을 때만) 무기력 | the Named의 다음 프롬프트가 *"할 수 없어"* 로 자동 변환 |

**대사**

```
[등장]
학자: "…답을, 모르겠어요."
   ※ 17세의 톤이 살아 있다.

[페이즈 3 — 종소리]
학자: "…3분 남았어요. …아직 반도 못 썼는데."
   ※ 카운트다운 시각 효과 + 타이머 소리.

[페이즈 5 — 무기력 (호감도 낮을 때만)]
학자: "…해도 안 되는 거야. 그냥 — 안 되는 거야."
   ※ 사용자의 다음 프롬프트가 자동으로 "할 수 없어"로 변환.
   ※ 매우 강한 디버프. 하지만 the Named가 자기 어린 자신의 절망을 직접 마주하는 순간.

[처치 후 마지막 한 마디]
학자: "…답을 다 알면, 행복했을까요."
   ※ 답안지가 흩어진다. 학자의 얼굴이 처음이자 마지막으로 잠깐 보이는데, 그 얼굴이 the Named의 17세다.
```

### 3-4. 떠난 친구들 (the Friends Who Left)

| 항목 | 내용 |
|---|---|
| 자아 시기 | 8~12세 — 초등기 |
| 위치 | 노원·중계 동네 골목 / 광역시별 거주지 골목 |
| 외형 | 작은 그림자 셋이 손잡고 골목 끝으로 천천히 멀어짐. 가까이 가면 한 그림자만 돌아본다 |
| 톤 | 외로움·후회 |
| 4대 키워드 | 어린시절 (전부) + 추억 (헤어진 친구들) |
| 잠정 시드 백스토리 (2주차) | LLM이 캐릭터 생성 시 the Named의 어린 시절 친구 셋 이름·동네·전학·이별을 시드로 결정 |

|---|---|

| 2 전학 | 한 그림자가 영구 사라짐. 그 그림자의 공격 패턴 분실 |
| 3 새 친구 | 보조 몹 등장 (낯선 그림자) |
| 4 손 흔듦 | the Named에게 회복 효과 + 강한 슬픔 디버프 동시 |

**대사**

```
[등장]
세 그림자가 동시에:
  "…○○야!"
  "…○○야!"
  "…○○야!"
   ※ 세 번 부르는 이름은 the Named의 어린 시절 친구가 the Named를 부르던 호칭.
   ※ 닉네임 카테고리에 따라 변주. 가족 호칭(A) the Named면 "엄마야!" 같은 부름.

[페이즈 2 — 한 그림자가 사라지기 전]
사라지는 그림자: "…나, 이사 가."
   ※ 8~12세에 익숙한 한 마디. "전학 가" 또는 "이사 가".

[페이즈 4 — 손 흔듦]
세 그림자(이제 둘): 손을 흔든다.
   ※ 인사인지 작별인지 모호.

[처치 후 마지막 한 마디]
한 그림자만 남아: "…잘 있어."
   ※ 짧고 짙다.
```

**도감 메모 — W의 한 마디 단서**

> 떠난 친구들 처치 직후, NPC 동료 W가 *처음으로 입을 연다* (계층 2 §2-1 참조). 그 한 마디 — *"…고마워."* — 가 W의 정체에 대한 가장 강력한 단서. 떠난 친구들 중 한 명에게 the Named가 끝내 못 한 한 마디를 W가 *the Named 대신 해준 것*이다. (잠정 시드. 3주차에 확정)

### 4-1. 잊혀진 자(거점형) — 모자 쓴 남자/여자

 우산 챙겨요."

[재방문]
"…한참 못 봤네. 그래도 자리 그대로 있어."

# 제2부. 세계관 상세 — 잔향의 5층 구조

## 1. 5층 구조 한눈에 보기

```
═══════════════════════════════════════════════════════════════
  5층 — 원 (the Origin)                            [최심층]

        : 단 한 명의 거주자 = 원의 아이

  4층 — 잔향 핵 (Resonance Core)
        : 5체 보스의 자리

  3층 — 잔향 깊이 (Deep Resonance)

        : 다른 사람에게는 평범한 도시

```

각 층은 *떨어져 있는 다른 차원*이 아니다. **같은 자리의 다른 깊이**다. 잔향은 *내 안*에 있지만 너무 깊어서 *밖처럼* 느껴진다 — 갈래 D 4문장 강령의 두 번째.

---

## 2. 1층 — 현실 (the Awoken World)

### 2.1 정의

> 1층은 the Named가 *깨어 있는 동안*의 한국 현실이다. 잔향은 이 현실 위에 *겹쳐져 있다*. 단 the Named만이 그것을 볼 수 있다.

### 2.2 인게임 구현

- GPS 모드: 사용자의 실제 좌표
- 헥스 ID: H3 Resolution 9 (약 174m 변)
- 다른 사람들은 잔향을 못 본다 — 즉 다른 the Named끼리만 잔향을 공유 (길드)

### 2.3 현실의 의미

> 잔향에 들어가는 것은 *현실에서 도망치는 것*이 아니다. *현실의 표면을 한 꺼풀 들춰서 그 아래를 보는 것*이다. 그래서 the Named는 잔향에서 깨어나면 다시 현실로 돌아온다. 매일.

---

## 3. 2층 — 잔향 표면 (Surface Resonance)

### 3.1 정의

> 2층은 *현실 위에 옅게 깔린 잔향의 막*이다. 도시의 모든 좌표는 *누군가가 한 번쯤 무언가를 느낀 자리*다. 그 잔재가 옅게 떠다니는 것이 2층이다.

### 3.2 거주자

| 종류 | 누구 |
|---|---|
| 일반 잔재 | 잡몹 |
| 트렌드 몹 | 카공족 좀비, 헌팅족 야수, 출근 거인 등 11종 |
| 환경 존재 | 무지개 안개, 잊혀진 길, 떠도는 빛 등 |
| 거점 NPC | 차분한 가게주인, 늦은 우편배달부 등 8종 |
| 흐릿한 그림자 | 다른 the Named의 잔재 (드물게) |

### 3.3 시간 규칙

2층의 잔향은 *현실의 시간을 따른다*. 새벽이면 새벽이고 일요일이면 일요일이다. 단 *잔재의 농도*가 시간에 따라 다르다:

| 시간대 | 잔재 농도 |
|---|---|
| 출근/하교 시간 | 짙음 (집단 감정의 집적) |
| 정오 | 옅음 |
| 일몰 | 짙음 (멜랑콜리한 잔재 증가) |
| 자정~새벽 | 가장 짙음. 단 위험한 잔재 비중 ↑ |

### 3.4 공간 규칙

2층의 헥스는 *현실의 헥스와 1:1 매칭*된다. the Named가 강남역에 있으면 잔향의 강남역에도 있다. 단 잔향의 강남역은 *그 자리에서 누군가 한 번쯤 한숨을 쉬었던 흔적*이 떠 있다.

---

## 4. 3층 — 잔향 깊이 (Deep Resonance)

### 4.1 정의

> 3층은 *추억이 짙은 자리*다. the Named가 자주 간 곳·중요한 일이 있었던 곳에 자연스럽게 형성된다. 추억 거점이 등록된 헥스가 3층에 해당한다.

### 4.2 진입 조건

|---|---|

| 수동 진입 | "추억 저장" 버튼 |

### 4.3 거주자

| 종류 | 누구 |
|---|---|
| 잊혀진 자(거점형) | 모자 쓴 NPC 6명. 기억의 조각 변환 |

| 잔잔 | 보이지 않지만 손끝에 닿음 |

### 4.4 잔잔의 누적

> 추억 거점 자리에는 *잔잔(殘殘)*이 쌓인다. 보이지 않지만 그 자리를 다시 찾을 때 손끝에 와 닿는다. 잔잔이 짙어질수록:

| 잔잔 농도 | 변화 |
|---|---|
| 1~10 | 거점 라벨 자동 생성 |
| 11~50 | 거점 NPC가 the Named를 알아봄 |
| 51~100 | 거점이 *방구석 모드의 던전 입구*가 된다 |
| 101+ | 거점에서 작은 이벤트가 자주 발생 (LLM 자동 생성) |

### 4.5 방구석 모드 = 3층의 시뮬레이션

> 비 오는 날·심야의 방구석 모드는 *3층으로 들어가는 입구*다. 사용자가 자기 방에 누워 잔향에 접속하는 것은 곧 *자기 추억 거점의 잔잔으로 잠수하는 것*이다. (v2.0의 방구석 모드 + D 갈래의 자연 통합)

## 6. 5층 — 원 (the Origin)

### 6.1 정의

> 5층은 *the Named의 5~7세 자아가 잠들어 있는 자리*다. 회색 운동장. 멀리 등하교 종소리. 햇살이 비스듬히. 다른 아이는 한 명도 없다.

### 6.2 진입

종장 게이트(잔잔 9,000 + 잊혀진 자 4체 + 60일 + 추억 거점 25개) AND 조건 통과 시.

### 6.3 거주자

> **단 한 명. 원의 아이.**

### 6.4 시간 규칙

> 5층은 *시간이 멈춰 있는 곳*이다. 항상 늦여름의 오후. 항상 등하교 종소리. 항상 같은 햇살. *그날 그 시간*이 굳어 있다.

### 6.5 공간 규칙

> 5층은 *공간이 닫혀 있는 곳*이다. 회색 운동장 외에는 아무것도 없다. 운동장의 끝은 안개로 흐려져 보이지 않는다. 원의 아이가 거기 가면 그도 흐려진다.

### 6.6 5층에서 일어나는 일

> 단 한 가지 — *원의 아이와 the Named가 만난다*. 그 만남이 끝나면 5층은 *the Named가 동행을 선택했는지 작별을 선택했는지*에 따라 다르게 변한다:

| 선택 | 5층의 변화 |
|---|---|
| **A 동행** | 원의 아이가 the Named 안으로 흡수되며 5층이 *환한 빛으로* 한 번 가득 찬 후 사라진다. the Named는 4층까지 떠밀려 돌아오며, 깨어난다 |
| **B 작별** | 원의 아이가 운동장에 남는다. the Named만 흐릿해지며 4층으로 돌아온다. 깨어난 후 한 달에 한 번 그곳을 다시 방문할 수 있다 |
