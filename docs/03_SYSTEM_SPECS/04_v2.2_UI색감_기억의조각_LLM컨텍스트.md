### 15.3 메인 스토리 3챕터 재설계 (4대 키워드 통합)

| 장 | 제목 | 키워드 | 감정 곡선 | 핵심 장면 |
|----|------|--------|----------|----------|
| **1장** | "이긴다고 외쳤던 날" | 어린시절 | 호기심 → 따뜻함 → 약한 위화감 | 튜토리얼. 첫 잔향에서 어떤 아이의 웃음소리가 들렸다 사라짐 |
| **2장** | "남은 것들의 도시" | 추억 + 꿈과 현실 | 몰입 → 의심 → 균열 | 추억 거점에서 만난 NPC가 *"너, 진짜야?"* 라고 묻는다 |
| **3장** | "내가 너였을 때" | 희생 + 어린시절 | 충격 → 슬픔 → 결단 | "목소리"의 정체 — *어린 시절의 나*. 마지막 선택: 데리고 갈 것인가, 두고 갈 것인가 |

### 15.4 "잃어버린 원" — 구체화 3문장 강령

> 1. **무엇을 잃었나**: 어른이 되며 *"이긴다!"* 고 외칠 수 있던 자신을 잃었다.
> 2. **왜 잃었나**: 잃어버린 것이 아니라, *어린 자신이 어른인 사용자를 위해 거기 남았다*. 자발적 희생이다.
> 3. **회복 가능한가**: 완전 회복은 불가능하다. 그러나 잔향에서 그를 만나고, 함께 시간을 보내고, 그의 조각을 가지고 돌아올 수 있다. *"전부 되찾을 수는 없지만, 잊지 않을 수는 있다."*

```
"밥 먹으러 들어오라고 누가 불렀고,

[00:30 — 현재의 사용자 방으로 페이드인]
"그때부터 빨강은 거기 있어.

"좋아. 시작해보자.

[01:30 — 첫 헥스 진입]
```

## 챕터 16. 회피 영역 + 참사 지역 표현 가이드라인

### 16.2 참사 지역 표현 — 4단계 표현 깊이

| Lv | 명칭 | 묘사 방식 | 적용 등급 |
|----|------|---------|----------|
| 0 | OFF | 일반 지역과 동일 | 옵트아웃 / 12세 |
| 1 | 정적 | "이상하게 조용하다", "바람도 멈춘 듯하다" | 12세 기본값 |
| 2 | 흔적 | "어디선가 작은 흐느낌이 들리는 것 같다" | **15세 기본값 (권장)** |
| 3 | 잔향(직접) | "이곳에서는 왠지 비명과 구슬픈 울음소리가 들린다" | 19세 기본값 |

**금지선 (모든 Lv 공통)**:
- 사건명·날짜·피해자 수·이름·신상 일체 X
- 가해자·책임자 추정 일체 X
- 정치적 비판·옹호 일체 X

### 16.3 위키/뉴스 링크 ⓘ 버튼

```
[모달]
이 장소에 대해 더 알고 싶다면

이곳에서는 실제로 일어난 일이 있어요.
게임에서는 직접 다루지 않지만,
알고 싶다면 아래 링크를 눌러주세요.

[위키백과에서 보기]    [뉴스 모음 보기]

[더 이상 보지 않기]    [닫기]
```

- 인앱 브라우저 사용
- 위키 한국어판 우선
- 뉴스는 연합뉴스·KBS·SBS 등 공공·메이저만 화이트리스트

### 20.4 비동기 처리 — 로딩 시간 최소화

```
사용자 턴 N 입력
   │
   ├─→ [즉시] 메인 LLM 호출 → 응답 반환 (목표 < 2초)
   │
   └─→ [백그라운드] Workers Queue
         └→ 요약 LLM → KV 갱신 (다음 입력 작성 중 완료)
```

**사용자 체감 로딩**: 0초 (요약은 다음 턴 *이전에* 완료, 평균 입력 작성 시간 7초 > 요약 LLM 시간 ~1.5초)
