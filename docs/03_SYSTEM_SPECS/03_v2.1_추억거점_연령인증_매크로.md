# LLM 기반 GPS 텍스트 MMORPG 게임 기획서 v2.1 추가 챕터 (9~13장)

> v1.0(스태미나·LLM 캐릭터 생성·GPS 헥스·Cloudflare 서버리스·3-tier 구독)과 v2.0(2D 타일맵 방구석 모드·세키로식 사망 시스템·스프린트 모드·한국 트렌드 지역 몹)에 이어, v2.1에서는 **추억 거점·사투 답파·세계관 스토리·연령 인증·소셜 로그인 + 스킬 흡수/매크로** 5대 시스템을 깊이 있게 설계한다. 본 챕터는 비개발자 1인 제작자 기준 *Vibe 코딩 가능 수준*으로 친절하게 작성되었다. 작성 기준 시점은 2026년 4월이다.

## 9장. 추억 거점 시스템 + 지역별 개인 스프린트 모드

### 9.1 챕터 개요와 핵심 가치 제안

**한 줄 요약**: 플레이어가 자주 방문한 현실 장소를 자동/수동으로 기록하고, 방구석 모드에서 그 장소를 "개인 가상 던전"으로 재구성한다. **"내 단골 카페가 던전이 되는" 경험**이 본 시스템의 정서적 핵심이다.

| 가치 축 | 설명 |
|---|---|
| 정서적 가치 | 일상 공간의 게임화 → 추억의 디지털 보존 |
| 운영적 가치 | 방구석 모드 콘텐츠 무한 확장 (개인별 절차 생성) |
| 보존적 가치 | 사용자가 떠난 곳도 게임 내에서 다시 방문 가능 |
| 차별화 | 기존 GPS 게임(Pokemon Go 등)이 못 하는 "개인 맞춤 절차 생성형 던전" |

### 9.2 방문 이력 기록 시스템 설계

#### 9.2.1 기록 트리거 (하이브리드)

| 모드 | 트리거 | 사용자 인지 |
|---|---|---|
| 자동 기록 | 동일 헥스 내 누적 30분 이상 머무름 | 기록 직후 토스트 알림 ("이곳을 추억으로 저장했습니다") |
| 수동 기록 | "추억 저장" 버튼 (HUD 우측 하단) 탭 | 즉시 기록 |
| 거부 | 옵트아웃 모드 활성화 시 양쪽 모두 비활성 | 첫 가입 시 명시적 안내 |

## 10장. 사투 답파 모드 + 희귀 강자 재전투 + 리더보드 통합

### 10.1 세키로 원본 시스템 분석

| 한국어 명칭 | 투기장 | 답파 |
| 잠금 해제 | 보스 처치 시 자동 | 메인 스토리 클리어 |

| 보상 | 없음 (연습용) | 신규 외형 + 신규 스킬 |

**핵심 통찰**: 세키로의 두 모드 모두 *외재적 보상이 매우 약하거나 없다*. "도전 자체가 보상". LLM 게임은 토큰 비용이 들기 때문에 **상징적 보상 + 리더보드**로 동기부여.

### 11.2 본 게임 자체 세계관 — "잔향 시스템"

#### 11.2.1 세계관 구조 (3계층)

```
현실계: 사용자가 사는 한국. 스마트폰이 *경계의 매개체*
잔향계: 현실의 감정·기억·집단 무의식이 데이터화된 평행 세계
원향계: 모든 잔향의 시초. 어린 시절 우리가 장난감을 싸우게 했던 그 *원초적 상상력*이 머무는 곳
```

#### 11.2.2 핵심 컨셉 3가지

| 컨셉 | 설명 |
|---|---|
| **1. 닉네임 = 영혼의 진짜 이름** | 무의식이 발화하는 한 마디로 운명이 결정 |
| **2. 장소 = 기억의 결정체** | 도시의 모든 좌표는 누군가의 기억과 욕망을 흡수한 "결정체" |
| **3. 스마트폰 = 매개체** | 사용자의 스마트폰은 잔향계와 통신하는 송수신기 |

### 11.3 메인 스토리 챕터 3개 구성안

| 챕터 | 제목 | 키워드 | 분량 | 클라이맥스 |
|---|---|---|---|---|

| 2장 | 「이름을 잃은 자들」 | 정체성·상실 | 10~20시간 | 잔향계가 무너지기 시작 |
| 3장 | 「원향에 닿다」 | 회귀·화해 | 10~30시간 | 원향계 진입, 어린 시절 자신과 마주함 |

### 13.4 스킬 흡수 시스템 — 본 게임의 핵심 매력

| 항목 | 값 |
|---|---|

| 흡수 슬롯 | **최대 3개** |
| 슬롯 교체 | 가능, 단 이전 스킬은 **영구 소실** |

#### 변형 예시

```
원본 스킬: 화염 거인 → "분쇄의 화염구"

LLM 변형:
  "그림자 화염의 비수"
  - 데미지: 원본의 60% (대신 출혈 추가)
  - 쿨다운: 8초 → 5초 (민첩 보정)
  - 묘사: "그림자 사이로 작은 불꽃이 솟아 적의 동맥을 꿰뚫는다"
  - 특수: 치명타 시 1회 더 발동 (운 보정)
```

### 13.5 WoW식 매크로 시스템
