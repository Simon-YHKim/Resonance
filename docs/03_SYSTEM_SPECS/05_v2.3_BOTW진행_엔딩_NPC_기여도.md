# 챕터 21. 탐험–스토리 자연 융합 — 잔잔 누적형 자연 진행

## 21.1 BOTW에서 가져올 것, 가져오지 말 것

| BOTW 원리 | 잔향 적용 여부 |
|---|---|
| **메인-사이드 경계 모호화** | ✅ 핵심 채택 — "스토리 모드" UI 절대 없음 |
| **호기심 미끼** | ✅ 채택 — GPS 헥스에 "잔향이 짙은 곳" 시각 큐 |
| **비선형 진행** | ⚠️ 부분 채택 — 어린시절 보스는 무조건 마지막. 4명의 잔재 보스는 자유 순서 |
| **정량 게이지 숨김** | ✅ 채택 — 잔잔 누적량은 the Named에게 "보이지 않음" |
| **메모리 12장 시스템** | ❌ 차용 안 함 — "수집해라"라는 압박 |
| **이정표 마커 NPC** | ⚠️ 최소 적용 — the Voice가 막혔을 때만 |

# 챕터 22. 기억할 순간 JSON 저장 시스템

## 22.1 핵심 개념

게임이 끝날 때, 화려한 정해진 컷씬 대신 *그 사람이 진짜 겪은 12~20개의 순간*을 다시 보여줍니다. 이 시스템 없이는 v2.3의 모든 챕터가 의미를 잃습니다.

## 22.2 자동 감지 — 3계층 필터

| 계층 | 방식 | 비용 |
|---|---|---|
| **L1 룰 트리거** | 정해진 이벤트 발생 시 무조건 후보 | ₩0 |
| **L2 키워드 필터** | "어머니/아버지/집/학교/그때/처음" 키워드 가중치 +2 | ₩0 |
| **L3 LLM 검증** | 후보가 일 5개 초과 시 Flash-Lite로 Top-K | ₩2/회 |

## 23.4 동기화 연출 — 한 장면 = 8초

| 시점 | 동작 |
|---|---|
| 0.0s | 이전 장면 페이드아웃(1.0s) |
| 1.0s | 이미지 페이드인(1.5s) |
| 2.0s | 1인칭 회상 텍스트 타이핑(글자당 50ms, 약 60자=3초) |
| 5.0s | 텍스트 완전 표시 |
| 7.5s | 다음 장면 트리거 가능 |
| 8.0s | 자동 다음 장면 |

**총 길이**: 16장 × 8초 = 약 2분 8초. 텍스트 길이에 따라 5~8분 가변.

## 23.5 비동기 처리 — 보스전 도중 백그라운드 생성

```

```

# 챕터 24. 어린시절 보스 설계 — 5체 점진적 접근

## 24.3 최종 보스 — **원의 아이 (The Child of Origin)**

### 외형 디자인

| 요소 | 묘사 |
|---|---|
| 키 | the Named 아바타의 1/3 (5~7세) |
| 얼굴 | the Named의 아바타가 **녹아 흘러내린 듯한** 흐릿함 |
| 옷 | 캐릭터 생성 시 무의식 어린시절 키워드에서 도출 |
| 배경 | 회색 운동장. 멀리 등하교 종소리. 다른 아이는 한 명도 없음 |
