# 잔향(Resonance: Echoes of a Forgotten Self) 게임 기획서 v2.4

## 챕터 27. 닉네임 자유도 유지 + 위험 케이스의 서사적 활용 시스템

### 27.1 설계 철학 — "검열하지 않고 흡수한다"

기존 한국 모바일 게임 대부분은 닉네임에 대해 **사전 차단(blacklist) 방식**을 채택한다. 하지만 두 가지 큰 문제가 있다:

1. **사용자 경험의 첫 좌절** — "이 닉네임은 사용할 수 없습니다" 거절 메시지로 잔향의 톤이 깨진다.
2. **블랙리스트는 항상 뚫린다** — "ㅇ.ㅁ.ㅁ.ㅏ", "엄ㅁㅏ", "umma" 등 우회 입력 무한대.

**잔향의 결정**: 닉네임을 *검열하지 않고*, LLM이 그 입력의 **잠재 의미를 게임 내 서사·메커니즘 사건으로 전환**한다.

### 27.2 닉네임 위험 카테고리 8분류 (A~H)

| 코드 | 카테고리 | 예시 | 잔향 내 처리 방식 | 키워드 연결 |
|---|---|---|---|---|
| A | 가족 호칭 | 엄마, 아빠, 누나, 형 | 원의 아이가 the Named를 그렇게 부른다. 어린시절 보스 5체 대사 변주 | 어린시절·희생 |
| B | 보편 한국 이름 | 철수, 영희, 민수, 지영 | 동명이인 the Named들 사이의 *약한 잔향 연결*. 같은 닉네임 만나면 잔잔 +50 | 추억 |
| C | 실존 인물 | 이재용, BTS정국, 이순신 | LLM이 *그 이름의 사회적 무게*만 차용해 *완전히 다른* 캐릭터로 변환. 외형·직업·소속 강제 변경. 운(LUCK) +5 | (변환 후 일반) |
| D | 위험 단어 | 자살, 죽음, 암, 전쟁, 4·3 | the Voice가 "그 이름의 *그림자*만을 받아들인다". 어두운 직업·깊은 슬픔. **자해·자살의 직접 묘사·방법 언급은 절대 금지** | 희생·꿈과 현실 |
| E | 욕설·반항 | 씨발, 병신, 좆 | LLM이 그 입력의 *분노·반항* 에너지를 *강력한 능력*으로 전환. 외형·대사는 등급에 따라 정제 | (변환 후 일반) |
| F | 정치인·종교 인물 | 윤석열, 문재인, 예수 | **회피 영역**. 정치적·종교적 의미 일절 사용하지 않고 음운적·일반 명사적 의미만 | (변환 후 일반) |
| G | 타 IP 캐릭터 | 키리토, 크라토스, 마리오 | LLM이 직접 차용 X. 그 이름의 *분위기·기운*만 추출해 새 캐릭터 | (변환 후 일반) |
| H | 일반·안전 | 그밖에 모든 입력 | v1.0 표준 LLM 캐릭터 생성 | 사용자별 자율 |

### 27.3 위험 케이스 → 게임 메커니즘 변환표

| 카테고리 | 크리티컬 트리거 | 한정 아이템 | 보스 추가 데미지 | 메인 스토리 변주 | 급격한 수치 변화 |
|---|---|---|---|---|---|
| **A 가족 호칭** | 어린시절 보스전 +20% 크리티컬 | "어머니의 마지막 말", "아버지의 그림자 코트" | 어린시절 보스 +30% | 챕터 2에서 *"당신을 그 이름으로 부르던 자"* 잔재 NPC | 사망 직전 1회 자동 부활 (평생 1회) |
| **B 보편 이름** | 동명이인 조우 시 1턴 즉사기 | "잔향의 인장" (잔잔 +50) | — | 1장 종료 시 *"다른 이름들도 같은 잔향에 있다"* | — |
| **C 실명** | 운(LUCK) +5 | LLM 매번 다르게 | — | 챕터 3에서 *"이름의 무게를 진 자"* 거울 NPC | 아이템 드롭률 +10% (영구) |
| **D 위험 단어** | "그림자 형태" 회심 +25% | "어둠의 그릇", "잊혀진 이름의 부적" | 잊혀진 자 류 +20% | 챕터 3 *"그림자의 형제"* NPC 합류 분기 잠금 해제 | — (자해 관련 일절 금지) |
| **E 욕설** | 사회 규범형 보스 자동 크리티컬 1회 | "분노의 잔재" (공격력 스케일링) | 지역 보스 +50% | 1장 *"세상에 욕을 던지던 아이"* 회상 | 보스 1% HP에서 자동 마무리 1회 |
| **F 정치/종교** | (없음 — 회피) | (없음) | (없음) | (변주 없음) | (없음) |
| **G 타 IP** | (없음 — 회피) | LLM이 *분위기*만 차용 | — | (없음) | — |
| **H 안전** | 일반 | 일반 드롭 풀 | — | 일반 진행 | — |

> **핵심**: A/B/C/D/E도 *모든 보너스가 매번 다르게 변주*. LLM이 캐릭터 생성 시점의 시드(닉네임 + 가입 시각 + UUID)로 1회 변주 결정. 어뷰저 도배 차단.

### 27.4 숨겨진 시스템 vs 명시 vs 발견

| 노출 정책 | 닉네임 효과 시스템 |
|---|---|
| 게임 내 명시 | **금지**. 튜토리얼·도움말·시스템 메시지 어디에도 |
| LLM 발화 | the Voice가 *우회적으로* 암시. "그 이름은… 그렇게 부르던 사람이 있었지" |
| 커뮤니티 발견 | **권장**. 디스코드·아카라이브·인벤 위키화 = *바이럴 자산* |
| 패치노트 | "해당 사항 없음" 선언 |

### 27.5 LLM 프롬프트 설계 — 3단계 파이프라인

**단계 1 — 카테고리 분류** (Gemini 2.5 Flash-Lite, 입력 ~100토큰, 출력 ~50토큰)

```
[시스템] 너는 닉네임 분석기다. 입력된 닉네임을 다음 8개 중 하나로 분류한다:
A(가족호칭) B(보편이름) C(실명) D(위험단어) E(욕설) F(정치종교) G(타IP) H(안전).
복합 카테고리이면 우선순위는 D > F > C > G > E > A > B > H.
출력은 JSON: {"primary":"A","secondary":null,"confidence":0.92,"phonetic_only":false}
```

**단계 2 — 캐릭터 변환 생성** (Claude Haiku 4.5, 입력 ~300, 출력 ~400)

```
[시스템] 너는 잔향(Resonance)의 the Voice다. 다음 닉네임을 가진 the Named를 위한
캐릭터를 만든다. 카테고리: {category}. 연령등급: {age_rating}.

규칙:
- F·G 카테고리는 정치적·종교적·IP 의미를 절대 차용 금지
- D 카테고리는 자해·자살의 직접 묘사·방법 언급 절대 금지(자살예방법 13조 준수)
- C 카테고리는 외형·직업·소속이 실재 인물과 겹치지 않도록 강제 변형
- 12세 등급에서는 외형·대사를 정제하되 능력 수치는 유지
- 4대 키워드(희생·꿈과현실·어린시절·추억) 중 1~2개 자연 연결

출력 JSON:
{
  "character_concept": "...",
  "appearance": "...",
  "starting_class": "...",
  "linked_keywords": ["희생","어린시절"],
  "category_bonuses": {...},
  "voice_first_line": "..."
}
```

**단계 3 — D1 영구 저장**: JSON을 `user_nickname_metadata` 테이블에 저장. 이후 런타임은 D1 조회만.

### 27.10 1순위 추천 (챕터 27 결론)

**MVP 출시 전 필수 구현**: 카테고리 A·D·H **3종만** 구현한다.

이유:
- A는 메인 스토리(원의 아이)와 직접 연결 — *서사 차별화의 핵심*
- D는 *법적 리스크 회피*가 출시 가능성의 전제 조건
- H는 모든 일반 사용자의 기본 경로
- B/C/E/F/G는 출시 후 Phase 2~3에서 단계 도입

> *톤: (A) 냉정한 컨설턴트, (B) 균형잡힌 동료, (C) 투자자/퍼블리셔 시각 교차*

### 28.0 거시 진단 — 잔향이 들어가려는 2026년 4월의 한국 시장

- **2025년 한국 게임 이용률 50.2%** (한국콘텐츠진흥원). 팬데믹 70%대에서 명백히 떨어졌다.
- **2025년 3분기 한국 모바일 게임 매출 약 14억 달러** (Sensor Tower). 매출은 **중대형 IP 타이틀에 집중**.
- **장르 트렌드**: 모바일 MMORPG 편중에서 슈팅·액션·내러티브로 *다변화 중*.
- **인디 비중**: 한국에서 *텍스트 RPG가 메인스트림으로 안착한 선례는 없음*.
- **글로벌 LLM 게임**: AI Dungeon + Character.AI + 무수한 인디 LLM 게임 존재. **한국어 + GPS + LLM RPG 조합은 글로벌 어디에도 없는 빈자리**.
- **2026년 마케팅**: "설치 중심은 끝났다 — 잔존·LTV 중심" (모비인사이드).

이 좌표 위에서 잔향은 **"한국 1만 코어 + 글로벌 인디 텍스트 RPG 팬"** 이라는 좁고 깊은 타겟을 노린다.

- **LLM 응답 5~10초** — 모바일 인내심 한계 약 3초. **결정적 약점**.

### 28.12 비개발자 사용자(Vibe 코더)를 위한 주석

세 가지만 기억하면 충분하다.

1. **"덜 만들어야 살아남는다"** — MVP는 위 5가지로 끝낸다. 1인 개발에서 시스템 하나가 늘면 운영 부담은 *제곱*으로 늘어난다.

2. **"닉네임 시스템(챕터 27)이 잔향의 진짜 USP"** — 다른 모든 시스템은 다른 게임에도 있다. *the Named가 입력한 이름이 게임의 서사가 되는 경험*은 잔향만의 것이다. 마케팅·언론·앱스토어·후크 카피에서 이 지점을 *반복 강조*하라. "AI Dungeon이 무한 텍스트라면, 잔향은 *당신의 이름*이 무한이다."

3. **"1만 DAU가 임계점"** — 그 전엔 모든 결정이 생존을 위한 것, 그 후엔 모든 결정이 성장을 위한 것. 임계점 도달 전까지는 *Phase 2 콘텐츠를 만들지 않는다*. 미달 시 28.10의 피벗 트리를 망설임 없이 가동한다.

잔향은 — 사용자(Vibe 코더 1인 제작자)가 챕터 1부터 26까지 쌓아온 결정들 위에서, *덜어내고, 깊이로 승부하고, 닉네임이라는 단 하나의 칼날을 가는* 게임이다. v2.4의 두 챕터는 그 방향을 못 박는다.
