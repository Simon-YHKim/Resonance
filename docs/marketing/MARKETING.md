5. **GPS RPG** (Phase 2 GPS 도입 시 활성화)

### 전체 설명 (스토어 4000자 한도, 800자 초안)

```
잔향(殘響: Echoes of a Forgotten Self)

당신의 이름은, 잔향이 된다.

닉네임 한 줄로 캐릭터의 인생이 만들어지는 한국향 LLM 텍스트 RPG.
'엄마'를 입력하면 그 이름으로 너를 부르던 사람의 그림자가 캐릭터가 된다.
'어둠'을 입력하면 그 이름의 그림자만을 받아들인 자가 된다.
같은 이름은 어디에도 없다 — 너의 발음, 너의 한 글자가 잔향이 된다.

[게임 플레이]
- 닉네임 입력 → AI가 너만을 위한 캐릭터를 만든다 (90초)
- 잊혀진 자와의 5턴 텍스트 전투 — 공격, 대화, 도망
- 누적된 잔향이 다음 거리의 무게가 된다
- 한국향 모던 멜랑콜리 톤 + 어두운 파스텔 비주얼

[안전 정책]
- 자해·자살 키워드는 분류만 하고 직접 묘사 절대 금지 (자살예방법 §19조의2 준수)
- 위험 시 자살예방상담 1393 안내 동시 노출

[현재 단계]
검증 프로토타입 (CBT 베타). 친구 5명 → 디스코드 50명 → 정식 출시 순서로
공개. 피드백은 디스코드 #피드백 채널 또는 simon@resonance.game.

[기술]
mock LLM (Phase 0) → Anthropic Claude Haiku 4.5 / Google Gemini Flash-Lite
(Phase 1) → Cloudflare Workers + AI Gateway 백엔드.
```

### 공유 메시지 템플릿

#### 친구에게 직접 (카톡·DM)

```
잔향 베타 1차 친구 시드
──────────────────────
당신의 이름은, 잔향이 된다.

[테스트 방법]
1. 갤탭/PC 브라우저에서 https://simon-yhkim.github.io/Resonance/ 접속
2. 90초 안에 첫 전투까지 가보고 한 줄 반응
3. 다른 닉네임 2~3개 더 시도 (엄마/민수/어둠/Simon 추천)

[원하는 피드백]
- 한 번 더 해보고 싶은가?
- 어떤 닉네임이 가장 인상 깊었나?
- 어디서 막혔나? (있다면)

[디스코드]
즉시 반응 → 디스코드 #피드백
https://discord.gg/resonance-beta

미리 고마워요. 시몬.
```

### 수집 KPI

5명 중:
- **D1 retention 35%+** = 5명 중 2명+ 다음 날 재방문 → ✅ Phase 1 진입 OK
- **"한 번 더?" 3명+** → ✅
- **90초 안에 첫 전투 5명** = ✅ 온보딩 통과
- **닉네임 2개 이상 시도 3명+** = ✅ 발견 트리거 작동

위 4개 중 3개 이상 ✅ → Phase 1 본 작업 진입 (실 LLM, GPS, 보스 5체)
2개 이하 → UX 개선 sprint 1회 더

### 시드 종료 후

- `RETRO.md` 작성 (정량 + 정성)
- 디스코드 시드 50명 캠페인 시작 (인벤·루리웹·트위터·Reddit)
