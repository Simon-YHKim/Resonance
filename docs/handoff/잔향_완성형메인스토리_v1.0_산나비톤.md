# 잔향(Resonance) 완성형 메인 스토리 v1.0

> **작성 시점:** 2026년 5월 4일 (Simon 외출 중)
> **분량:** 약 8~12시간 (산나비 톤)
> **구조:** 90초 튜토 + 1장(5장면) + 2장(12장면) + 3장(15장면) + 종장(8장면) + 엔딩 (6 × 16장 = 96장 시안)
> **톤:** K-멜랑콜리 모던 / 위로하는 게임 / 미완성의 보존

---

## 0. 산나비 톤 작성 원칙

### 0.1 산나비가 10시간으로 명작인 비결 — 잔향에 적용

| 산나비 비결 | 잔향 적용 |
|---|---|
| 모든 장면이 *진행* | 묘사·설명 X. 행동·대사로만 |
| 한 마디로 캐릭터 박음 | 보스마다 *마지막 한 마디* 박힘 |
| 시각적 카타르시스 | 종장 *종이 펼치기* 시각 클라이맥스 |
| 짧지만 압축 | 한 장면 = 한 의미 |
| 한국적 정서 정수 | 모남·꿈자국·잔잔 어휘 |
| 사족 없는 결말 | 엔딩 16장 = 1인칭 짧은 회상 |

### 0.2 작성 형식

```
[장면 N - 제목]
[배경: 시공간·분위기 한 줄]
[등장: 캐릭터들]

[지문]
"대사" — 화자
"대사" — 화자

[액션·인터랙션]

[셋업/페이오프 노트] - 어느 단서를 박거나 회수하는지
[이미지 리스트] - 이 장면에서 필요한 이미지
```

## 1. 90초 튜토리얼 (5장면)

### 90초-1 — 검은 화면

```
[배경: 검은 화면. BGM 침묵.]

[지문]
화면이 — 천천히, 천천히 밝아진다.

[화면에 떠오르는 한 줄]
"…기억나요?"

[3초 침묵. 글자가 흐려진다.]

[셋업: 단서 1 — "기억나요"]
[이미지: 검은 화면 + 흰 글자]
```

### 90초-2 — 빨강과 파랑

```
[배경: 두 손이 화면에 등장. 어린아이의 손.]
[한 손엔 빨강 색종이. 한 손엔 파랑 색종이.]

[the Voice — 첫 발화]
"…그때, 외쳤어요.
 빨강이 좋아요, 라고. 
 또는 — 파랑이 좋아요, 라고."

[두 손이 — 빨강과 파랑을 부딪힌다. 가위바위보처럼.]

[셋업: 단서 2 — "외쳤어요" / 단서 3 — "빨강·파랑"]
[이미지: 어린아이 두 손이 빨강·파랑 색종이를 부딪히는 정지 이미지]
```

### 90초-3 — 부른 목소리

```
[배경: 화면이 — 갑자기 정적.]
[빨강·파랑 색종이가 멈춘다.]

[the Voice]
"…근데, 누군가 — 부르더군요.
 그 이름. 처음 부르던, 그 이름."

[부르는 소리는 — 들리지 않는다. 단지 *부른 자국*이 보인다.]
[색종이를 든 두 손이 — 한 쪽은 빨강을 데려가고, 한 쪽은 파랑을 책상 위에 두고 간다.]

[the Voice]
"빨강 하나는 — 데려갔고,
 파랑 하나는 — 책상 위에 두고 갔어요."

[빈 책상. 파랑 색종이 하나만 남는다.]

[셋업: 단서 4 — "처음 부르던 그 이름" / 단서 5 — "당신만 거기 없어요"]
[이미지: 빈 책상 위 파랑 색종이 하나]
```

### 90초-4 — the Named의 첫 입력

```
[배경: 책상 이미지가 흐려진다. 검은 화면.]

[the Voice]
"…이 자리, 알아요?"

[the Named의 입력창이 활성화된다. 첫 입력.]

[가능한 입력 시안 — 예시]
"…기억나요" / "…아뇨" / "…모르겠어요" / "…빨강과 파랑이 뭐예요"

[유저의 입력에 따라 the Voice 응답 변주]

[the Voice — 어느 입력이든 부드럽게]
"…그러면, 잘 왔어요."

[셋업: 단서 6 — "잘 왔어요"]
[이미지: 검은 화면 + the Named 입력창]
```

### 90초-5 — 잔향으로의 진입

```
[배경: 검은 화면이 — 옅은 회색으로 변한다.]

[the Voice]
"여기는 — 잔향이에요.
 잠들기 직전과 잠든 사이의 자리예요.
 한 번도 가본 적 없는 줄 알았는데 —
 자주 와봤던 자리예요.
 
 잘 지내셨어요?"

[화면이 — the Named의 방으로 전환된다. 늦은 밤.]

[1장 시작]

[Long con 시작점 — 6개 단서 모두 박힘. 1회차 12% 인지.]
[이미지: 회색 → the Named의 방 (낮은 천장, 침대, 작은 창)]
```

## 2. 1장 — 자기를 잃었음 자각 (5장면, 약 45분)

### 1-1 — 늦은 밤의 침대

```
[배경: 11시 47분. the Named의 방. 침대 위. 누워서 천장.]
[자기정당화 모놀로그가 자동 재생됨 — 약 2분 30초]

[the Named의 내면 독백 — 음성 자동 재생. 자막도 표시.]

  "…아, 등이 좀 뻐근하네...
  
   …내일도 9시까지인데. 빨리 자야 되는데.
   
   …오늘 회의에서 — 그 말 안 하길 잘했다.
   괜히 분위기만 어색해졌을 거야.
   다들 그냥 — 빨리 끝내고 싶어 하는데...
   나만 한마디 하면 또 — 시간 끄는 사람 되는 거지.
   
   …근데, 김 대리는 — 어떻게 그렇게 자기 의견을 잘 말하지.
   나도 — 옛날엔 그랬는데.
   
   …아니, 됐어. 
   지금은 — 이게 더 편해. 다들 이렇게 살잖아."

[the Voice가 첫 등장 — 채팅창 한 줄]
"…아셨어요?"

[the Named의 모놀로그가 멈춘다. 천장을 본다.]

[the Voice]
"…천장 너머에, 무엇이 있는지."

[the Named의 입력창 — 자유 입력]
[유저가 무엇을 입력해도 the Voice가 받아들임]

[셋업: the Named의 자기정당화 = 게임 시작점]
[이미지: 11시 47분의 어두운 방, 천장]
```

### 1-2 — 첫 잔향

```
[배경: 천장의 한 모서리가 — 살짝 흐려진다.]

[the Voice]
"…저기, 보세요. 작은 그림자."

[침대 머리맡에 — 작은 그림자가 떠다닌다. 첫 길거리 몹.]
[몹 이름: "오늘 못한 한 마디"]
[외형: 흐릿한 작은 형체. 입 모양만 살짝.]

[the Voice]
"…저게, 첫 잔향이에요. 한 번, 만나볼래요?"

[튜토리얼 — 첫 LLM 텍스트 전투]
[the Named의 입력창]

[유저가 입력 → LLM 응답 → 작은 전투]
[처치 후]

[모남 조각 (소) 획득]
"오늘 못한 한 마디의 한 조각"
"2026.05.04 23:48 / 오늘 회의에서 못 한 한 마디"

[the Voice]
"…잘 했어요. 
 이게 — 잔향에서 자기를 만나는 방식이에요.
 작은 한 조각씩, 다시."

[페이오프 시작: 모남 조각 누적의 첫 카드]
[이미지: 천장 모서리에서 떠다니는 작은 흐릿한 그림자, "오늘 못한 한 마디" 외형]
```

### 1-3 — 첫 추억 거점 등록

```
[배경: 화면이 — the Named의 일상 풍경으로 전환]
[주의: 이 장면은 *게임 안*이 아니라 *게임 메커니즘 안내*]

[the Voice]
"…이제, 한 자리, 가봅시다.
 자주 가는 곳이 있죠?
 그 자리에 — 잔향이 짙어요."

[GPS 활성화 시]
- 사용자가 자주 머무는 헥스 자동 인식
- 첫 추억 거점 자동 등록
- 거점 이름은 LLM이 변주 (예: "당신이 자주 가는 카페", "자주 내리는 정류장")

[GPS 미활성 시 — 방구석 모드]
- 자기 방이 추억 거점 1번
- 자동 등록

[the Voice]
"…여기, 알아요. 자주 — 와요?"

[유저 응답에 따라 변주]

[the Voice]
"…기억해둘게요. 이 자리."

[셋업: 추억 거점 시스템 학습]
[이미지: 카페 또는 방의 일러스트, 거점 등록 UI]
```

### 1-4 — 거인이 보인다

```
[배경: the Named가 잔향 안에서 — 첫 도시 헥스 진입]
[헥스: 강남역 또는 광역시별 동급. 사용자 좌표 기반.]

[the Voice]
"…멀리, 뭔가 보여요?"

[화면 끝에 — 거대한 형체가 보인다. 빌딩 12층 높이.]
[정장 실루엣. 머리 자리에 형광등 무더기.]

[the Voice]
"…저게, 첫 잊혀진 자예요.
 이름을 가진 사람만 — 만나야 해요. 가까이 가요."

[보스 1 등장 — 남겨진 거인]
[the Named가 다가가는 도중 일반 길거리 몹 1~2회 등장 가능]
[몹: 출근 거인 (작은 분신), 카공족 좀비 등 그 헥스의 시간대에 맞게]

[셋업: 보스 1 = 첫 메인 보스]
[이미지: 강남역 빌딩 사이로 보이는 거대한 거인 실루엣, 형광등 머리]
```

### 1-5 — 보스 1 남겨진 거인

```
[배경: 강남역 (또는 광역시별) 헥스. 12층 높이의 거인.]

[거인 등장 한 마디]
거인: "…늦었어, 또."

[전투 시작]
[페이즈 1 — 빌딩 그림자 낙하]
[페이즈 2 — 형광등 깜빡임 (스턴)]
[페이즈 3 — 출근길 군중 소환]
거인: "…여기, 다 너 같은 사람들이지."

[페이즈 4 — 명함 폭격]

[처치 직전 — 거인의 외형이 흐려진다]
[형광등 무더기 사이로 — 작은 빛 하나가 새어나온다]

[the Voice]
"…뭔가, 떠오르려고 해요."

[LLM이 the Named의 직장 시드에서 한 장면 호출]
[흑백 톤으로 잠깐 떠오름]

  "…20대 어느 회식. 당신이 한 사람을 너무 챙겼던 그 밤.
   다음 날 — 다른 동료가 한 마디 했죠. 
   '너무 친한 척하지 마. 어색해.'
   그날 이후 — 당신은 동료를 일정 거리 이상 안 들였어요."

[처치 직후]
거인: (한층 부드러운 톤) "…그래서, 행복했었나요."

[화면이 천천히 흐려진다. 잠깐 색이 돌아왔다 다시 흑백으로.]
[보스가 흩어진다 — 단 *조각 하나*가 the Named에게 남는다.]

[모남 조각 (대) 획득]
"동료에게 너무 정 들이려 했던 자기 한 조각"
"2026.05.04 / 30대의 어느 회식, 잘려나간 정"

[the Voice]
"…좋은 사람들이었어요. 잊지 말아요."

[1장 클로징]

[the Voice]
"…여기까지가, 첫 자리였어요.
 다음에, 또 만나요."

[게임 자동 종료]
[다음 날 — the Named가 다시 게임을 켤 때, 자연스럽게 2장 진입]

[페이오프: 90초 튜토 빨강·파랑 단서가 *직장 풍경*과 처음 접합]
[이미지: 거인 4페이즈 외형, 처치 시 작은 빛, 흑백 회식 장면]
```

## 3. 2장 — 조각의 누적과 패턴 (12장면, 약 2~3시간)

### 2-1 — W의 합류

```
[배경: 다음 세션. the Named가 게임을 다시 켰다.]
[방. 같은 침대. 단 — 천장에 한 명 더 있다.]

[the Voice]
"…W예요. 같이, 가도 돼요?"

[W의 외형]
[나이 모호. 회색 후드. 두 손 항상 주머니에. 채팅창에는 "…"만 출력.]

[the Named가 W를 받아들임]
[W 합류]

[페이오프: 4대 키워드 *희생* 매핑된 첫 동료]
[이미지: 회색 후드를 입은 W의 정면 실루엣]
```

### 2-2 — 자유 진행 안내

```
[배경: 도시 헥스 지도가 화면에 뜬다.]

[the Voice]
"…이제, 자유롭게 — 가고 싶은 곳에 가요.
 어디에든 — 잔향이 있어요.
 
 그리고 — 두 자리, 가야 할 곳이 있어요.
 한강. 또는 — 학원가."

[헥스 지도에 두 빨간 점 — 한강과 학원가]
[사용자 자유 선택]

[셋업: 보스 2·3 자유 순서]
[이미지: 한국 대도시 헥스 지도, 두 빨간 점]
```

### 2-3 — 한강 둔치 가는 길 (길거리 몹 시퀀스)

```
[배경: the Named가 한강 둔치로 향한다.]
[가는 도중 — 헥스마다 길거리 몹]

[몹 1 — 출퇴근 헥스]
"지하철 3호선의 그림자"
한 마디: "…내릴 자리가 — 자꾸 헷갈려."
처치 후 the Voice: "…매일 같은 자리 가는데, 매일 새롭게 헷갈려요. 다들 그래요."
모남 조각 (소): "오늘 또 헷갈린 한 조각"

[몹 2 — 점심시간 헥스]
"결정장애 망령"
한 마디: "…김치찌개? 된장찌개?"
처치 후 the Voice: "…오늘도 정하지 못했네요. 괜찮아요."
모남 조각 (소): "오늘 점심을 못 정한 한 조각"

[몹 3 — 한강 도착 직전]
"러닝메이트 환영"
한 마디: "…같이 뛰어… 같이…"
처치 후 the Voice: "…혼자 뛰는 게 더 편할 때도 있어요."
모남 조각 (소): "혼자 뛰는 게 좋은 한 조각"

[셋업: 길거리 몹 메커니즘 정착]
[이미지: 각 몹의 외형 3종, 한강 둔치 풍경]
```

### 2-4 — 흐르는 그림자 등장

```
[배경: 한강 둔치. 일몰. 강물 위에 두 그림자.]
[한 그림자가 — 천천히 다른 쪽에서 멀어진다.]

[the Voice]
"…저기, 두 그림자."

[보스 2 등장 — 흐르는 그림자]

[그림자 등장 한 마디]
그림자: "…아직, 보이는구나."

[전투 시작]

[페이즈 1 — 두 갈래 분열]
왼쪽 그림자: "…따라와."
오른쪽 그림자: "…놓아 줘."
[the Named가 둘 중 하나 선택. 어느 쪽도 정답 X.]

[페이즈 2 — 비]
[화면이 흐려진다. 시야 차단 + 스태미나 회복 정지.]

[페이즈 3 — 강 물결]
[잊혀진 메시지가 떠올랐다 사라진다.]
"…잘 지내?"
"…미안해."

```

4. 한 시간 후. 작은 결과물.

   "사장님이 알아본다."

   "선물했다. 받은 사람이 — 한참 봤다."

   "*…공방에 다녀요.*"

9. 사진을 인스타에 올린다.
   "처음 댓글이 — 사장님이었다."

10. 가끔 어린 자신을 떠올린다.
    "그 아이도 — 손으로 뭔가 만드는 걸 좋아했다."

    "그러나 — 화제가 늘었다."

12. 어느 주말. 작은 플리마켓.
    "팔린 건 — 셋. 그러나 셋이다."

13. 일 년 후. 단골 손님.
    "*다음번 작품 기다려요.*"

14. 동료의 한 마디. *너 — 좀 변했어.*
    "*…그래?*"

15. 어느 저녁. 공방. 늦게까지.
    "조용한 음악. 손의 움직임."

16. 책상 위. 빨강 하나. 파랑은 — 어딘가. 그 사이에 작은 손길로 만든 무엇.
    "셋이 — 어울려 있다."
