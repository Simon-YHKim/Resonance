   → 항상 한국어

## 5. 잔향의 5대 영혼 (절대 흐려선 안 됨)

빌드 도중 *현실적 타협*이 들어올 때 — 다음 5개는 **절대 지켜야 합니다**:

```
영혼 1. 메타피직스 갈래 D — 잠들기 직전 30분의 게임
영혼 2. the Voice = 어린 자신의 수호자 자아 (4단계 정체 공개 곡선)
영혼 3. 보스 = 모남(잘려나간) + 꿈자국(되어보지 못한) 조각
영혼 4. 위로하는 게임 — 인정 + 동행, 판단 X, 폭력 X
영혼 5. 미완성의 보존 — *간직이 곧 행복* (박준형 정수)
```

빌드 결정 시 *영혼 4번 (위로하는 게임)*에 위배되면 — Simon에게 즉시 확인 요청.
예: 보스를 *적*으로 표현하거나, 사용자 점수에 *경쟁 요소*를 추가하거나, 자기학대 메커니즘 등.

## 6. 잔향의 핵심 시스템 — 빌드 전 필수 이해

### 6.1 5층 세계관

```
1층: 현실 (the Named만 잔향을 봄)
2층: 잔향 표면 (옅은 잔향이 깔린 풍경)
3층: 잔향 깊이 (잔잔이 짙어진 추억 거점)
4층: 잔향 핵 (5체 보스의 자리)
5층: 원 (회색 운동장 — 종장의 자리)
```

### 6.2 5체 보스 — 시간 역행 구조

```
보스 1: 남겨진 거인 (30~40대 직장 자기)
보스 2: 흐르는 그림자 (20대 사랑 자기)
보스 3: 미루는 학자 (17~19세 꿈 자기)
보스 4: 떠난 친구들 (8~12세 우정 자기) ← 결정적 변곡점 (점수 우세 축 결정)
보스 5: 원의 아이 (5~7세 첫 자기) ← 종장
```

### 6.3 6엔딩 시스템

```
첫 선택: 동행 / 작별
두 번째 선택: 개화(開花) / 여전(如前) / 항로(航路)

= 6엔딩
  엔딩 1 동행+개화 (회사 화장실 → 화실)
  엔딩 2 동행+여전 (출근길 지하철 → 일상)
  엔딩 3 동행+항로 (주말 카페 → 부업)
  엔딩 4 작별+개화 (한강 둔치 → 새 일)
  엔딩 5 작별+여전 (마을버스 → 일상) ⭐ 가장 한국적
  엔딩 6 작별+항로 (동네 공방 → 손작업)
```

### 6.4 NPC 동료 4인

```
W (침묵) — 4대 키워드 *희생*
꼬마 R — 4대 키워드 *어린시절*
양수 — 4대 키워드 *꿈과 현실*
노 선생 — 4대 키워드 *추억*
```

### 6.5 인게임 워딩 사전 (절대 변경 X)

```
잔향(殘響) — 게임 정체성
원 / 원의 자리 — 5층 세계관 종착점
이름을 가진 사람 (the Named) — 사용자 캐릭터
목소리 (the Voice) — 메타 캐릭터
잊혀진 자 (the Forgetter) — 5체 보스 + 거점 NPC
모남 조각 (대 / 소) — 잘려나간 자기
꿈자국 조각 — 되어보지 못한 자기
잔잔(殘殘) — 잔향이 짙어진 상태
사라짐 — 게임 사망
건너기 — 헥스 이동
남은 자리 — 추억 거점
```
