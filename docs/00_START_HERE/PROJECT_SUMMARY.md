# PROJECT_SUMMARY — 잔향 3분 요약

> Claude Code가 가장 먼저 읽는 문서. 3분 안에 잔향의 모든 핵심을 잡습니다.

---

## 1. 한 문장으로 정의

**잔향은 — 사회에 맞춰지며 잘려나간 자기와 한 번도 되어보지 못한 자기를, 한 조각씩 다시 만나며, 내 안에 살아 있는 모든 가능성을 다시 그리는 이야기다.**

장르: LLM 기반 GPS 텍스트 MMORPG
플랫폼: 모바일 메인 + 웹(PWA) 보조
타겟: 한국 1만 코어 + 글로벌 인디 텍스트 RPG 팬
톤: K-멜랑콜리 모던 (위로하는 게임)

## 4. 게임 동력 — RPG로서의 핵심 등식

```
보스 1체 처치 = 모남 또는 꿈자국 조각 1개 획득
              = 자기 정체성의 한 면 알아감
              = 종합 점수 누적

5체 모두 처치 = 5개 조각 모임
              = 자기 정체성의 한 그림이 보이기 시작
              = 종장 진입 가능

종장         = 모든 조각이 원의 자리에서 한 그림으로 풀림
              = 카타르시스 정점
```

## 5. K드라마 4단 구조

```
기 (1장) — 자기를 잃었음을 깨닫는 단계 (45분 압축)
승 (2장) — 조각이 모이며 패턴이 보이는 단계 (25~30일)
전 (3장) — 결정적 반전과 정체성의 흔들림 (30~40일)
결 (종장) — 모든 조각이 한 그림이 되는 카타르시스 (30~60분)
```

## 6. 5체 보스 — 시간 역행 구조

```
[1] 남겨진 거인        — 30~40대, 강남, 모남 70% / 꿈자국 30%
[2] 흐르는 그림자      — 20대, 한강, 모남 60% / 꿈자국 40%
[3] 미루는 학자        — 17~19세, 종로, 모남 30% / 꿈자국 70% ← 꿈자국 메인
[4] 떠난 친구들        — 8~12세, 동네 골목, 모남 65% / 꿈자국 35%
                         ⭐ 2/3 지점 = 결정적 변곡점
[5] 원의 아이 (최종)   — 5~7세, 원의 자리, 모남 50% / 꿈자국 50%
```

## 7. 6엔딩 시스템 (NEW v1.2)

**2단 분기 = 동행/작별 (잔향 안) × 개화/여전/항로 (깨어난 후) = 6 엔딩**

### 3엔딩 네이밍 (시안 B 한자 시적)

| 엔딩 | 한자 | 의미 |
|---|---|---|
| **개화(開花)** | 開花 | 꿈을 향해 사표를 던지는 결단 |
| **여전(如前)** | 如前 | 다시 정을 맞으며 출근하는 받아들임 |
| **항로(航路)** | 航路 | 현실적으로 가능한 새 길을 찾는 절충 |

### 6엔딩별 깨어나는 장소

| # | 엔딩 | 장소 | 시간 |
|---|---|---|---|
| 1 | 동행+개화 | 회사 화장실 칸 | 평일 점심 직후 |
| 2 | 동행+여전 | 출근길 지하철 | 평일 아침 |
| 3 | 동행+항로 | 주말 카페 | 토요일 오전 |
| 4 | 작별+개화 | 한강 둔치 | 평일 새벽 일출 직전 |
| **5** | **작별+여전** ⭐ | 마을버스 | 평일 아침 (가장 한국적) |
| 6 | 작별+항로 | 동네 공방 | 평일 저녁 |

### 종합 점수 메커니즘

```
3축 누적 (룰 기반, LLM 호출 0):
  - 개화_axis (도약형): 보스 정면 돌파, 결단형 어휘, 빠른 진입
  - 여전_axis (수용형): 거점 재방문, 차분한 대화, 신중함
  - 항로_axis (절충형): 매크로 다양, 다른 the Named 교류, 새 헥스

2/3 지점 (보스 4 처치 시) → 우세 축 50% 이상 결정
보스 4 처치 후 → 몹들의 행동이 우세 축에 맞춰 변화
종장까지 → 변경 가능 (어렵지만)
종장 → 사용자 마지막 선택 (시스템 점수와 다른 엔딩 선택 가능)
```
