## 1.7 세계관 바이블 v0.1 (2026년 4월 29일)

### 메타피직스 갈래 D — "기억의 꿈"

**4문장 강령**:
1. **누구의 것인가** — the Named 본인의 것이다. 다른 누구의 것도 아니다.
2. **어디에 있는가** — *내 안*에 있다. 그러나 너무 깊어서 *밖처럼* 느껴진다.
3. **진짜인가** — 꿈이 진짜 같듯이 진짜다. 깨고 나면 흐려지지만 *겪은 것은 사라지지 않는다*.
4. **왜 가야 하는가** — 아무도 대신 가줄 수 없으니까. *직접 걸어가서 만나야 비로소 이름을 되찾는다*.

### the Voice — 해석 2 (어린 시절의 수호자 자아)

**정체**: the Named 안에서 어린 자신을 잊지 않으려 남아 있던 수호자 자아. 원의 아이를 지켜온 자.

**화법**: C 혼합형 (평소 잔잔한 산문체 + 결정적 순간 시적 단문)

**4단계 정체 공개 곡선**:

| 시기 | 공개도 | 단서 수 | 핵심 표현 |
|---|---|---|---|
| 1장 | 0% | 3개 | "그 이름, 오래 들어보네요" |
| 2장 | 20% | 4개 | "…안 돼, 거기까지 가면 안 돼" (사라짐 직전 단 한 번) |
| 3장 | 50% | 5개 | "우리"라는 단어 처음 사용 |
| 종장 | 100% | 명시 | "여기서부터는, 당신이에요" |

**분기 엔딩에서**:
- A 동행: 원의 아이와 함께 the Named 안으로 흡수
- B 작별: 원의 자리에 함께 남음

### 빨강과 파랑

| 색 | 정체 |
|---|---|
| 빨강 | the Named (그날 데려간 자아) |
| 파랑 | 원의 아이 (책상 위에 두고 간 자아) |
| 부른 목소리 | 명시 X. 닉네임 카테고리 A에서만 톤 변주 |

### 90초 튜토리얼 단서 6개

1. "기억나요" (이미 알고 있는 사이의 어휘)
2. "외쳤어요" (본인이 그 자리에 있었던 듯)
3. 빨강·파랑 중 어느 쪽이 남았는지 정확히 앎
4. "당신만 거기 없어요"
5. "처음 부르던 그 이름"
6. "그 이름, 오래 들어보네요"

> 1회차 평균 단서 인지율 12% / 회차 회상 시 90%+

## 1.8 캐릭터 도감 v0.2 (2026년 4월 29일)

### 5계층 36종

**계층 1. 중심 인물 (3종)**
- the Named (사용자 본인, 어른의 자아)
- the Voice (수호자 자아)
- 원의 아이 (5~7세 잠든 자아, 최종 보스)

**계층 2. 메인 NPC 동료 (4종)** — 백스토리 잠정 시드, 3주차 확정 예정
- W (침묵의 W, 희생) — 4번 보스 후 한 마디
- 꼬마 R (어린시절) — 7세 어린이, 3장 끝 작별
- 양수 (꿈과 현실) — 가짜 밝음 청년, 종장 직전 작별
- 노 선생 (추억) — 70대 노인, 종장 진입 시 떠남

**계층 3. 잊혀진 자 — 5체 보스 (5종)**
- 남겨진 거인 (30~40대, 강남)
- 흐르는 그림자 (20대, 한강)
- 미루는 학자 (17~19세, 종로)
- 떠난 친구들 (8~12세, 동네 골목)
- 원의 아이 (5~7세, 원의 자리)

**계층 4. 거점 + 스토리 NPC (8종)**
- 잊혀진 자(거점형, 모자 쓴 6명)
- 차분한 가게주인
- 늦은 우편배달부
- 꿈꾸는 학생
- 졸린 경비
- 떠도는 음악가
- 한 마디 노인
- 흐릿한 그림자 (다른 the Named의 잔재)

**계층 5. 트렌드 몹 + 환경 존재 (16종)**
- 일반 잔재 / 카공족 좀비 / 헌팅족 야수 / 출근 거인 / 러닝메이트 환영 / 야식 망령 / 세일 광인 / 시험 망상 / 결혼 광기 / 짜파게티 환영(일요일 한정) / 운명석 야상 / 무지개 안개 / 잊혀진 길 / 떠도는 빛 / 메아리 짐승 / 작은 이름 짓는 자

### 세계관 5층 구조

```
5층 — 원 (5~7세 회색 운동장, 원의 아이만)
4층 — 잔향 핵 (5체 보스의 자리)
3층 — 잔향 깊이 (추억 거점, 잊혀진 자 NPC, 방구석 모드)
2층 — 잔향 표면 (GPS 도시 헥스, 트렌드 몹)
1층 — 현실 (한국, the Named만 잔향을 봄)
```

### 7대 물리법칙

1. **the Named의 법칙** — 본인만 잔향을 봄
2. **시간의 법칙** — 1·2·3·4층은 현실 시간 따름. 5층만 시간 멈춤
3. **공간의 법칙** — 1·2·3층은 현실 1:1. 4층은 핵 5개. 5층은 닫힘
4. **사라짐의 법칙** — 죽음 = 깨어남
5. **잔잔의 법칙** — 사라져도 자리는 남음
6. **기억의 조각의 법칙** — 잊혀진 자는 주인을 기다림
7. **the Voice의 법칙** — 깨달을 때까지 말하지 않음

> **법칙이 깨지는 단 한 순간**: 종장에서 원의 아이가 입을 여는 순간. 잔향 전체의 카타르시스 정점.

## 2.3 보스 정체 진단 — Simon의 결정적 발언

1. 시간의 먼지 (단순 망각)
2. 어른의 합리화 (자기에게 한 거짓말)

5. 자아의 분열

**Simon의 결정**: **2 + 3 결합** + *"위로하는 게임"*

## 2.4 Simon의 *진짜* 정서 — 4월 29일 가장 중요한 발화

> *"어떤 사회의 일부분이 되었다면 모난 부분들이 정해 맞아 틀에 맞춰지는 과정들이 있다고 생각해. 그 모난 부분들은 정말 모나서 없어졌어야 할 수도 있고 자신만의 개성이었을 수도 있어. 그 모든 것들이 다 떨어져 나갔지만 어쨌거나 나의 한 부분이었어"*

**이 한 단락에서 박힌 결정**:

1. **가치 판단 X** — *좋은 것이었는지 나쁜 것이었는지 모름. 단 그게 너의 일부였음을 인정*
2. **회복 X** — *떨어져 나간 것은 돌아오지 않음. 한 번 더 보고 다시 두고 옴*
3. ***모남*이라는 정확한 단어** — 명사 *4대 키워드*보다 형용 *모남*이 더 정확

1. 본 문서 (`잔향_종합정리_v1.0_컨텍스트보존.md`)의 *제5부 빠른 참조 테이블*
2. 본 문서의 *제2부 메인 주제 진행 중 작업*
3. 필요시 개별 v1.0~v2.4 + 세계관 바이블 + 캐릭터 도감 참조

## 6.2 Simon이 잊지 말아야 할 4가지

이 게임의 *영혼* 4가지. 디자인 결정이 흐려질 때 다시 박을 것:

1. **메타피직스 갈래 D — 기억의 꿈** (환각 X, 평행세계 X, 데이터 X)
2. **the Voice 정체 = 수호자 자아** (어린 자신을 지켜온 자, 어린 자신 본인 X)
3. **모남이 떨어져 나간 자리** (덮인 본래 기억 X, 회복 X, 가치 판단 X)
4. **위로하는 게임** (싸구려 위안 X, 훈계 X, 인정 + 동행 O)

## 6.3 다음 대화에서 Claude가 바로 시작할 수 있는 시그널

Simon이 다음 대화 열 때 이 중 어느 한 마디면 *컨텍스트 즉시 복원*:

- *"4개 결정 답할게"* → 메인 주제 박기 진행
- *"2주차 시작"* → 5체 보스 시드 백스토리 작업
- *"3주차 시작"* → NPC 동료 아크 작업
- *"종합 정리에서 다시 시작"* → 본 문서를 토대로 다음 작업 결정
- *"~을 어떻게 다뤄야 할까"* → 본 문서 참조 후 답변
